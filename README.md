# vocaloids
"https://miku-api.vercel.app/vocaloids" <br>
this is the link ^ <br>
its set up as an array with objects<br>
here is an example 

```json
[
    {
        "name": "",
        "codename":"",
        "introduction":"",
        "Version":"",
        "image":""
    }
]
```
Currently it has 86 diffrent vocaloids <br>
All of the original VOCALOID and VOCALOID 2 <br>
